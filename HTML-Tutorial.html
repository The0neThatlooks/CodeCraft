Let’s start with the basics!

This is a div with a paragraph (p) inside of it.

“Div” stands for division. Think of them as boxes. It’s commonly
used to group stuff together.

“p” stands for paragraph. It’s used to add text to your website!

Links, Photos & Comments

Links (<a>) in HTML are used to create clickable hyperlinks.

Images (<img>) in HTML are used to add images to your webpage via the “src” attribute that
stands for source. You can copy an image address but also import pictures to use.

Comments (<!-- -->) in HTML are notes in the code for developers that aren’t displayed in the browser.

Lists:

In HTML, lists are used to organise and structure content in a sequential or unordered manner.

This is an ordered list (ol) with list items (li) inside of it.

Ordered lists are used to present items in a specific
order, usually with numerical or
alphabetical indicators.

This is an unordered list (ul) with list items (li) inside of it.

Unordered lists are used to present items without any
inherent order, typically with bullet points as markers.

Tables:

HTML tables are used to organise and display data in a structured format on a web page.

This is a table with table rows (tr), table headers (th)
and table data (td)

“<table>”is the container for the entire table.

“<tr>” is the container for the entire table.

“<td>” is the container for the entire table.
