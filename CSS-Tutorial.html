Let’s start with the basics!

Element Selector

Element selectors in CSS are used to target and apply
styles to specific HTML elements on a webpage. An element
selector selects all occurrences of a particular HTML
element and allows you to define how those elements
should be styled.

Class Selector

Class selectors in CSS are used to select and apply
styles to HTML elements that have a specific
class attribute. A class attribute allows you to
assign a name to one or more HTML elements,
and then the class selector is used in CSS
to target and style those elements.

Pseudo Element Selector

Pseudo-elements in CSS allow you to style parts
of an element without having to add extra HTML markup.
They are classified by double colons (::). There's also
Pseudo Classes (:) that target and style elements
based on their state or position in the document, such
as user interaction, structural position, or input status.

Box Model

The box model defines how elements on a web page are structured and how space is
distributed around and within them. The box model consists of four main components:
Content, Padding, Border, Margin
